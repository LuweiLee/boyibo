# 搏一搏 (Boyibo) - AI生成的博彩推荐社区

> **🤖 本项目完全由AI生成** - 使用Claude AI助手从零开始构建，包括前端代码、后端架构、UI设计、数据存储等所有功能模块。

**项目口号：搏一搏，单车变摩托！** 🏍️

## 📋 项目简介

搏一搏是一个现代化的博彩推荐与微社区平台，为用户提供专业的体育赛事分析、推荐服务以及社区交流功能。

### ✨ 核心功能

- **🏠 首页**：平台统计数据展示，活跃用户、推荐准确率等关键指标
- **📊 推荐页面**：专业体育赛事分析推荐（NFL、英超、NBA、德甲等）
- **💬 社区页面**：用户交流讨论，分享经验心得
- **👤 个人页面**：用户个人信息管理（已隐藏，专注核心功能）

### 🎯 特色功能

- **智能昵称管理**：自动保存用户昵称，刷新页面无需重复输入
- **真实点赞统计**：防重复点赞，跨设备数据同步
- **实时时间显示**：准确显示发帖时间（刚刚、X分钟前、X小时前等）
- **云端数据同步**：使用Cloudflare R2存储，数据安全可靠
- **响应式设计**：完美适配桌面端和移动端

## 🛠️ 技术栈

### 前端技术
- **构建工具**: Vite
- **核心技术**: 原生JavaScript (ES6+)
- **样式**: CSS3 + 渐变设计
- **模块化**: ES Modules
- **存储**: localStorage + 云端同步

### 后端架构
- **云存储**: Cloudflare R2 Object Storage
- **API代理**: Cloudflare Workers
- **数据格式**: JSON
- **CORS解决方案**: Worker代理模式

### 部署平台
- **前端部署**: Cloudflare Pages
- **CDN加速**: Cloudflare全球网络
- **域名解析**: Cloudflare DNS
